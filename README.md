# 📘 Frontend — Application de Gestion Administrative

## 🧩 Présentation

Ce projet représente le  **frontend d’une application de gestion administrative** , développée avec Angular.

Il fournit une interface moderne et responsive permettant d’interagir avec une API backend développée en  **Spring Boot** .

## ⚙️ Stack technique

* **Angular** (framework frontend)
* **TypeScript**
* **Tailwind CSS** (styling utilitaire)
* **shadcn-ng** (composants UI modernes)
* **RxJS** (gestion des flux asynchrones)
* **Node.js / npm** (gestion des dépendances)

## 📁 Prérequis

Avant de commencer, assure-toi d’avoir installé :

* Node.js (v18+ recommandé)
* npm ou pnpm
* Angular CLI
