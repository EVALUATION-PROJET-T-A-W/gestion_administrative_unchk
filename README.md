README — Frontend (Angular + Tailwind + shadcn-ng)

Ce projet est le frontend d’une application de gestion administrative, développé avec :

Angular 
Tailwind CSS
shadcn-ng 
TypeScript

Il consomme une API backend (Spring Boot).

📦 Stack technique
Angular
Tailwind CSS
shadcn-ng UI
RxJS
TypeScript
Node.js / npm

⚙️ Installation

1. Installer les dépendances
npm install
2. Lancer le projet
ng serve

Puis ouvrir :

http://localhost:4200
